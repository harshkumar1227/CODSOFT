# Simple Rule-Based Chatbot

A beginner-friendly chatbot project built using HTML, CSS, and JavaScript.

This chatbot responds to user messages using predefined rules and pattern matching techniques with if-else conditions.

---

## Features

- User input handling
- Rule-based chatbot responses
- Chat interface UI
- Auto scrolling chat
- Responsive design
- Beginner-friendly project

## How It Works

The chatbot:
1. Takes user input
2. Converts input to lowercase
3. Matches keywords using if-else conditions
4. Returns predefined responses

Example:

```javascript
if(input.includes("hello")) {
   return "Hello! How can I help you?";
}
```
